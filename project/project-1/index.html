Overview

The project goal is to make a personal site hosted on the internet. The three main components of this site are Hugo static site generator and wowchemy Academic template and GitHub hosting.

The site home page contains my profile picture, biography, information about education, skills, experience and my posts.

Hugo

A static site generator takes your content, applies it to a template, and generates an HTML based static site. It’s excellent for blogs and landing pages.

  • Benefits:

    • Quick deployment
    • Secure (no dynamic content)
    • Fast load times
    • Simple usage
    • Version control

To start with Hugo first i downloaded the last version of Hugo. Then i created a new repository cloned from Wowchemy academic template. After cloning the repository and creating a new repository to host the site. I generated the site using Hugo.

Wowchemy

The Wowchemy Hugo academic template help to create the site and supply the neded file to us so we can customize it and work on it.

Simply cloned the template as a new repository.

Github

GitHub free hosting is the main allowed us to host the site on the internet. by creating a repository in the form of username.github.io. In this repository the coded of the site.

This project helped me to understand how GitHub hosting work. How important to have a personal blog that push you to post your work and ideas.
